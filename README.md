# NotilusSchedulerMobile

NotilusScheduler masaüstü (WinForms) + NotilusSchedulerWeb (React) uygulamalarının birebir mobil klonu.
Stack: React Native + Expo (TypeScript). DevExpress YOK.
Backend: **YENİ API YAZILMAZ** — NotilusSchedulerWeb.Server'daki mevcut ASP.NET Core API aynen kullanılır (aynı database, aynı JWT auth, aynı endpoint'ler).

## Kurulum ve Çalıştırma

1. `npm install` (ilk seferde)
2. `src/config.ts` içindeki `API_BASE_URL`'i API'nin koştuğu makinenin LAN IP'sine göre ayarla (örn. `http://192.168.1.34:5240`).
3. NotilusSchedulerWeb.Server'ı LAN'dan erişilebilir başlat:
   `dotnet run --urls "http://0.0.0.0:5240"`
4. `npx expo start` → telefonda **Expo Go** ile QR kodu okut (telefon + PC aynı Wi-Fi'da olmalı).

- `src/screens/MainShell.tsx` — web sidebar'ın mobil karşılığı (üst bar + soldan menü). Renkler `useThemeColors()` ile temaya bağlı: koyu #1e2433 / #2d374b / #b9c3d7 / aktif #2d374e, açık tema web `App.css` `.theme-light` birebir. Segoe MDL2 ikonları mobilde yok; şimdilik yalnızca etiket.
- `src/theme.ts` — AppTheme mobil karşılığı (masaüstü `Logic/AppTheme.cs` + web `App.css` `.theme-light`).
